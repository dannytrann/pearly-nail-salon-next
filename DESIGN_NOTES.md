# Pearly Nails & Spa - Design System Notes

## Typography (from Square Site)
- **Headings**: Playfair Display (weight 400)
- **Body Text**: PT Serif (weight 400)
- **Font Scale**: Base 20px with 1.107 multiplier
- **Line Heights**: Titles 1.1, Body 1.3
- **Letter Spacing**: Titles -0.01 to -0.02em

## Original Color Palette
- Primary: #144676 (deep blue)
- Accents: #1fd8cc, #15cfe4 (teal)
- Backgrounds: #d3edf5, #d0f3ef, #e5ebeb (light)

## New Custom Site Colors
- **Primary**: #e67e22 (orange) - for our custom group booking site
- Keep the clean, professional aesthetic

## Layout & Components
- Squared button style with 100px border radius
- Cards with 8px border radius
- Images with 16px border radius
- 3-column grid for services (responsive)
- 12px grid gaps
- Min header height: 80px
